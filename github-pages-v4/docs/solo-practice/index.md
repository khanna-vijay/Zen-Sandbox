# 🧘 Solo Practice — Improv Drills You Can Do Alone
> *You don't need a stage, a scene partner or a class to get better — you need reps. These are the ones you can do alone, today.*

A field guide of **30 solo improv exercises** you can practise by yourself at home, organised into **7 skill themes**. Each one is a self-contained drill with a clear objective, step-by-step play, variations to grow into, and a square infographic you can screenshot and take with you.

!!! tip "How to use these"
    Pick a theme you want to strengthen, set a timer, and **commit fully** — solo practice only works if you actually say it out loud, on your feet. Do a little, often. Every page has a *How to play*, *Why it works*, and a coach's tips section.

!!! abstract "Every page, the same shape"
    **🎯 Objective** (what the drill is for) → **▶️ How to play** (numbered steps) → **🔁 Variations** (ways to grow it) → **💡 Why it works** → **🎓 Coach's tips**. Every page also carries a square infographic you can screenshot and take with you.
