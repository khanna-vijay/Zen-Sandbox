## 🧭 How to read it

Every concept lives at the intersection of three axes — the map draws the first two:

| Axis | Question | Values |
|---|---|---|
| **Domain** | *In relation to whom / what?* | Self → Partner → Scene → Ensemble → Audience |
| **Layer** | *At what level of abstraction?* | **Principle** (Why) → **Skill** (What) → **Technique** (How) |
| **Stage** | *At what level of development?* | Novice → Adv. Beginner → Competent → Proficient → Master |

!!! abstract "The central thesis"
    Mastery is not knowing *more* techniques — it is executing the *same* techniques governed by *deeper principles.*
    **We learn bottom-up** (Technique → Skill → Principle) and **perform top-down** (Principle → Skill → Technique).

- 💎 **Principle — the *value*.** The belief that defines what "right" means (unobservable; held as conviction).
- 🧠 **Skill — the *brain*.** The judgment of *which* technique to deploy, *when*, and *how much*.
- 🎯 **Technique — the *muscle*.** A discrete, repeatable, observable action you can drill.

> The whole framework rests on **Layer 0 — the Safety & Consent Container** (shown at the hub): psychological safety is the environment every other domain lives inside.

> *Architect compelling scenes in real time — using both engines (Game & Narrative) and knowing which the scene wants.*
